# دليل التشغيل المحلي

## الهدف

هذا الدليل يشرح طريقة تشغيل السكربتات العملية الأولى في المشروع لتوليد تقارير ونشرات ومسودات تذكير من ملفات CSV محلية.

السكربتات تعمل محليًا فقط ولا تقوم بأي اتصال خارجي أو إرسال رسائل أو تعديل بيانات المصدر.

## المتطلبات

- Python 3.9 أو أحدث.
- وجود ملفات البيانات داخل مجلد `samples/`.
- وجود القوالب داخل مجلد `templates/`.
- لا توجد مكتبات خارجية مطلوبة حاليًا. المشروع يستخدم مكتبات Python القياسية فقط.

## هل النظام مفعّل على أجهزتي؟

وجود الملفات في GitHub لا يعني أنها تعمل تلقائيًا على اللابتوب أو الآيفون.

حتى تكون مفعّلة فعليًا على جهازك، يجب أن تكون:

1. نسخة المشروع موجودة على اللابتوب.
2. Python مثبت على اللابتوب.
3. يتم تشغيل السكربتات يدويًا من الطرفية Terminal / PowerShell.
4. تظهر ملفات داخل مجلد `output/`.

إذا لم يظهر مجلد `output/` بعد التشغيل، فهذا يعني أن السكربتات موجودة في GitHub فقط ولم تعمل على جهازك بعد.

## طريقة التأكد السريعة من التفعيل

من داخل مجلد المشروع على اللابتوب، شغّل:

```bash
python scripts/run_local_workflow.py --date 2026-05-12 --exchange-rate 15000
```

بعدها تحقق من وجود هذه الملفات:

```text
output/daily-report-2026-05-12.md
output/price-list-2026-05-12.md
output/payment-reminders/2026-05-12/summary.md
output/inventory-movement-2026-05-12.md
output/audit-log.csv
```

إذا ظهرت هذه الملفات، فهذا يعني أن النموذج المحلي يعمل على جهازك.

## تشغيل كل شيء بأمر واحد

هذا هو الأمر الأفضل للتشغيل اليومي المحلي:

```bash
python scripts/run_local_workflow.py --date 2026-05-12 --exchange-rate 15000
```

## نتيجة التشغيل الشامل

سينشئ الأمر:

```text
output/daily-report-2026-05-12.md
output/price-list-2026-05-12.md
output/payment-reminders/2026-05-12/summary.md
output/inventory-movement-2026-05-12.md
output/audit-log.csv
```

## ماذا يفعل التشغيل الشامل؟

- يولد تقرير نهاية اليوم.
- يولد نشرة الأسعار اليومية.
- يولد مسودات تذكير العملاء المتأخرين.
- يولد تقرير حركة الأصناف: مطلوب وناقص / بطيء / راكد للتصريف.
- يسجل ما تم توليده داخل سجل تدقيق محلي.
- لا يرسل أي شيء.
- لا يتصل بأي خدمة خارجية.

## تشغيل تقرير نهاية اليوم فقط

من داخل مجلد المشروع، شغّل الأمر التالي:

```bash
python scripts/generate_daily_report.py --date 2026-05-12
```

## نتيجة تقرير نهاية اليوم

سيتم إنشاء ملف تقرير داخل مجلد `output/` باسم مشابه:

```text
output/daily-report-2026-05-12.md
```

## تشغيل نشرة الأسعار اليومية فقط

يمكن تشغيل مولد نشرة الأسعار مع إدخال سعر الصرف مباشرة:

```bash
python scripts/generate_price_list.py --date 2026-05-12 --exchange-rate 15000
```

أو بدون تمرير سعر الصرف، وسيطلبه السكربت يدويًا:

```bash
python scripts/generate_price_list.py --date 2026-05-12
```

## تشغيل تذكير العملاء المتأخرين والأرصدة فقط

```bash
python scripts/generate_payment_reminders.py --date 2026-05-12 --overdue-days 4
```

## تشغيل تحليل حركة الأصناف فقط

```bash
python scripts/analyze_inventory_movement.py --date 2026-05-12
```

ينتج:

```text
output/inventory-movement-2026-05-12.md
```

هذا التقرير يوضح:

- الأصناف المطلوبة وكمّيتها قليلة.
- الأصناف بطيئة الحركة.
- الأصناف الراكدة أو التي تحتاج عرض تصريف.
- الأصناف التي تحتاج مراجعة بيانات.

## القوالب التجارية المتاحة

تمت إضافة قوالب يمكن استخدامها يدويًا لاحقًا، ولا يتم إرسالها تلقائيًا:

```text
templates/limited-quantity-announcement.md
templates/clearance-offer-announcement.md
templates/supplier-reorder-request.md
```

هذه القوالب مخصصة لـ:

- إعلان كمية محدودة للزبائن.
- إعلان عرض خاص لتصريف أصناف مختارة.
- طلب إعادة توريد من المورد.

## تشغيل الاختبارات المحلية

بعد أي تعديل على السكربتات، يمكن تشغيل الاختبارات التالية:

```bash
python tests/test_local_workflow.py
```

إذا نجحت الاختبارات، ستظهر الرسالة:

```text
All local workflow tests passed.
```

## سجل التدقيق المحلي

عند تشغيل `run_local_workflow.py`، يتم إنشاء أو تحديث:

```text
output/audit-log.csv
```

يسجل هذا الملف:

- وقت توليد التقرير.
- وقت توليد نشرة الأسعار.
- وقت توليد مسودات التذكير.
- وقت توليد تقرير حركة الأصناف.
- أن التشغيل تم محليًا.

ولا يسجل:

- كلمات مرور.
- أكواد تحقق.
- أسرار.
- مفاتيح API.

## ما لا تفعله السكربتات

- لا ترسل رسائل واتساب.
- لا ترسل SMS.
- لا ترسل بريدًا إلكترونيًا.
- لا تتصل بالإنترنت.
- لا تجلب سعر الصرف تلقائيًا.
- لا تدخل إلى أي حساب.
- لا تغيّر ملفات البيانات الأصلية.
- لا تطبع فواتير.
- لا ترفع أي ملف إلى iCloud أو أي خدمة خارجية.
- لا تنشر على السوشيال ميديا.

## ملاحظة أمان

هذه السكربتات مخصصة كبداية محلية آمنة. قبل استخدام بيانات حقيقية، يجب تجربتها على بيانات وهمية والتأكد من أن النتائج صحيحة.
