# ربط جرد المواد مع النشرة

## الملف الذي تم فحصه

تم فحص ملف جرد مواد بصيغة Excel يحمل ورقة رئيسية باسم:

```text
جرد المواد
```

## الأعمدة المكتشفة

```text
اسم المادة
الوحدة الثانية
عامل تحويل الوحدة
الكمية
السعر
القيمة
```

## معنى الأعمدة

- `اسم المادة`: اسم الصنف داخل برنامج المحاسبة.
- `الوحدة الثانية`: غالبًا كرتونة أو وحدة التخزين المستخدمة.
- `عامل تحويل الوحدة`: عدد الكروزات داخل الكرتونة أو عامل التحويل.
- `الكمية`: كمية الصنف الموجودة في الجرد.
- `السعر`: سعر الصنف داخل تقرير الجرد.
- `القيمة`: إجمالي قيمة الكمية.

## طريقة استخدامه مع النشرة

ملف الأسعار يبقى مصدر السعر المعتمد للنشرة.

ملف الجرد يستخدم فقط لتحديد الأصناف الموجودة فعليًا في المستودع.

النظام يجب أن:

1. يقرأ ملف الأسعار.
2. يقرأ ملف جرد المواد.
3. ينشر فقط الأصناف الموجودة في الجرد بكمية أكبر من صفر.
4. يستبعد الأصناف الموجودة في الأسعار لكنها غير موجودة في الجرد.
5. يولد تقريرًا بالأصناف الموجودة في الجرد وغير موجودة في ملف الأسعار.
